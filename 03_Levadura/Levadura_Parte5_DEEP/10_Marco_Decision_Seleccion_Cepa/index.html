    Marco de decisión para selección de cepa¶

    Este documento define cómo elegir cepa como decisión sistémica: mosto + proceso + perfil sensorial + limitaciones reales.
    Elegir cepa es elegir un comportamiento, no un nombre.

    Entrada mínima (lo que tenés que saber antes de elegir)¶

    Variables primarias¶

    • Perfil de azúcares del mosto (fermentabilidad)
    • OG / alcohol objetivo
    • Temperatura de fermentación disponible
    • Perfil sensorial buscado (limpio vs carácter: ésteres/fenoles)

    Variables secundarias¶

    • Floculación deseada (y claridad objetivo)
    • Reutilización prevista (repitch sí/no, cuántas generaciones)
    • Estabilidad requerida (biológica/sensorial/shelf life)
