    ESTRUCTURA_REPO — Árbol canónico (Quelonio Pages V2)¶

    Este archivo es la “vista de inventario” del repo para evitar duplicados (ej. dos 10_*.md en la misma carpeta).

    Regla operativa¶

    Antes de crear/nombrar un archivo nuevo, se debe: 1) Consultar este árbol (o el inventario del módulo/carpeta objetivo). 2) Elegir un número libre dentro de esa carpeta. 3) Crear contenido y linkear desde el índice correspondiente.

    Alcance del árbol¶

    • Incluye: docs/ (estructura navegable MkDocs)
    • Excluye: site/, .git/, .obsidian/, node_modules/, etc.
