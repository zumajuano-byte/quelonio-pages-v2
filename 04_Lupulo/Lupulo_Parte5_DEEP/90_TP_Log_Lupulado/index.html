    TP — Log de Lupulado (Diseño + Ejecución + Resultado)¶

    Este TP estandariza el registro del lupulado por lote para poder: - repetir resultados, - comparar lotes, - y diagnosticar fallas (oxidación, vegetal, hop creep, pérdidas).

    0) Identificación del lote¶

    Campo Valor
    Lote / ID
    Estilo
    Fecha
    Volumen al KO (L)
    Volumen a fermentador (L)
    Operador

    1) Objetivo sensorial (1 frase)¶

    Campo Valor
    “Una frase” objetivo
    Amargor percibido bajo / medio / alto
    Aroma objetivo (nota líder + 2 secundarias)
    Mouthfeel objetivo seco / medio / alto / “juicy”

    2) Materia prima (lúpulos)¶

    Registrar por variedad y lote para trazabilidad.

    Variedad Forma (pellet/cono/extracto) Lote %AA Año cosecha (si está) Observación (frescura/uso)

    3) Arquitectura de lupulado (capas)¶

    Marcar qué “capas” usás y por qué.

    Capa Sí/No Intención
    Amargor base (hervor o extracto)
    Late hopping (opcional)
    Whirlpool / Hopstand
    Dry hop post-ferment
    Dry hop durante ferment (biotransf.)

    4) Ejecución por etapa (registro operativo)¶

    4.1 Hervor (amargor)¶

    Campo Valor
    Tiempo de hervor real (min)
    Adición(es) (variedad / g / min)
    IBU objetivo (si aplica)
    Observaciones

    4.2 Whirlpool / Hopstand¶

    Campo Valor
    Temp de entrada (°C)
    Temp final (°C)
    Tiempo total (min)
    Agitación / recirculación
    Adición(es) (variedad / g)
    Observaciones (vegetal/astringencia)

    4.3 Dry hop (una o varias cargas)¶

    Si hacés 2 cargas, completá 2 bloques.

    Carga #1 | Campo | Valor | |---|---| | Momento | durante ferment / post-ferment | | Día/etapa (ej. day 2, post FG) | | | Temp cerveza (°C) | | | Tiempo contacto (h) | | | Total (g) | | | g/L (sobre volumen real) | | | Variedades (detalle) | | | Método (abierto/cerrado) | | | Observaciones | |

    Carga #2 | Campo | Valor | |---|---| | Momento | durante ferment / post-ferment | | Día/etapa | | | Temp cerveza (°C) | | | Tiempo contacto (h) | | | Total (g) | | | g/L (sobre volumen real) | | | Variedades (detalle) | | | Método (abierto/cerrado) | | | Observaciones | |

    5) Control de oxígeno (DO / práctica)¶

    Campo Valor
    ¿Se hizo purga/presurización? sí / no
    ¿Proceso cerrado? sí / no
    DO medido (si tenés)
    Observaciones (riesgo oxidación)

    6) Riesgos y mitigaciones (marcar lo que aplique)¶

    Riesgo Sí/No Acción aplicada
    Hop creep descanso diacetilo / control temp / tiempo
    Diacetilo plan de control / medición / descanso
    Vegetal / hop burn bajar tiempo / bajar temp / mejorar separación
    Pérdidas por absorción ajuste g/L / separación / técnica
    Astringencia revisar temp/tiempo/pH/proceso
    Oxidación purgas / cerrado / empaque

    7) Rendimiento (pérdidas y separación)¶

    Campo Valor
    Volumen pre-lúpulo (L)
    Volumen post-lúpulo (L)
    Pérdida atribuible a lúpulo (L)
    Método de separación crash / transferencia / filtración / otro
    Observaciones

    8) Resultado y feedback loop (lo que manda)¶

    Campo Valor
    Aroma (intensidad 1–5)
    Amargor (1–5)
    Astringencia/rasposo (1–5)
    “Verde/vegetal” (1–5)
    Estabilidad (días hasta caída percibida)
    Qué repetir (1–2 líneas)
    Qué cambiar (1–2 líneas)
