# 08 — Recetas & Specs (DEEP)

## Propósito del módulo
Convertir “receta” en un sistema controlable: cambios trazables, decisiones explícitas y feedback loop con sensorial.

## Qué vas a operar con este DEEP
- Definir o ajustar una receta con criterio (targets y trade-offs).
- Registrar un cambio como “ticket” (qué cambió, por qué, qué se espera).
- Convertir feedback sensorial en cambios concretos de receta/spec.
