# 05 — Preparación para envasar (entrada desde 07)

## Si venís desde Fermentación/Maduración (07)
Antes de envasar, confirmá que ya está cerrado el final del proceso:

- diacetilo resuelto (ideal: prueba forzada)
- transferencias sin oxígeno
- cerveza fría/estable (si aplica)

➡️ Volver a: [07/30 — Transferencias, cold crash y salida](../../07_Fermentacion_Maduracion/DEEP/30_Transferencias_cold_crash_y_salida.md)

---

## Objetivo (en simple)
Llegar al envasado con la cerveza:
- limpia (sin diacetilo),
- estable (temperatura/claridad según el plan),
- y con el menor oxígeno posible.

---

## Checklist “Listo para envasar” (mínimo)
### A) Cerveza lista
- [ ] Densidad final estable
- [ ] Diacetilo OK (ideal: prueba forzada)
- [ ] Claridad / sedimento dentro de lo esperado para el estilo
- [ ] Temperatura de envasado definida (y estable)

### B) Proceso listo
- [ ] Líneas y recipiente purgados con CO₂ (si aplica)
- [ ] No hay “puntos donde chupe aire” (conexiones, válvulas, mangueras)
- [ ] Plan de carbonatación definido (método y objetivo)
- [ ] Plan de control de oxígeno definido (aunque sea “práctico”)

---

## Oxígeno: el enemigo #1 en esta etapa
Regla simple: en frío, casi todo lo que sale mal empieza con oxígeno.

➡️ Documento guía: [20 — Control de oxígeno (DO/TPO)](20_Control_oxigeno_DO_TPO.md)
