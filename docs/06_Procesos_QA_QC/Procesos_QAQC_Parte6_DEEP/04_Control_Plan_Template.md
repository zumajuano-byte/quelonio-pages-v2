# 04_Control_Plan_Template

> Propósito: “una hoja” que resume **qué se controla, dónde, con qué método, cada cuánto, quién y qué hacer si falla**.

## Encabezado
- Producto/estilo:
- Proceso: (Brewhouse / Fermentación / BBT / Packaging)
- Versión del control plan:
- Fecha / responsable:
- Riesgo fuente: (FMEA / HACCP / mapa de proceso + inputs/outputs)

## Reglas de “no burocracia”
- No incluir variables sin **acción definida**.
- No medir si el equipo no está calibrado.
- Mantener 1 control plan por familia de cerveza + anexos por tecnología/pack.
