# Glosario (simple)

- **Índice**: página “menú” de un tema.
- **DEEP**: “parte profunda”. Explicación más detallada y procedimientos.
- **Checklist**: lista corta para no olvidarse pasos clave.
- **Placeholder**: marcador de “esto se completa después”.
- **Oxidación**: entrada de oxígeno que arruina frescura.
- **DO (oxígeno disuelto)**: oxígeno dentro de la cerveza.
- **OG / FG**: densidad inicial / densidad final.
