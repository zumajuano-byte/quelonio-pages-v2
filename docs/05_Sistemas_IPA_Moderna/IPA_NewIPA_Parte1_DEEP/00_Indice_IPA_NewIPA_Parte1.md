## Cómo usar esta Parte 1
- Si tu problema es “no logro tropical real”: empezá por **Tioles**.
- Si tu problema es “se me apaga el aroma”: **Fermentación/retención**.
- Si tu problema es “raspa / hop burn / astringe”: **Polifenoles/mirceno**.
- Si tu problema es “mismo proceso, resultados distintos”: **Variabilidad/trazabilidad**.
