# Implicancias: receta y proceso (hot-side aplicado)

## 1) Receta: el hot-side no flota en el vacío
Para que el hot-side “funcione” como se espera, receta y agua deben acompañar:

- Si diseñás hot-side “suave” pero el agua tiene filo extremo, el amargor se percibe más duro.
- Si diseñás hot-side “aromático” pero el cuerpo es muy bajo, la IPA queda filosa.
- Si diseñás hot-side correcto pero oxidás en transferencia, perdés el trabajo.

## 2) Proceso: decisiones que te ahorran retrabajo
- Definir “capas” de hot-side por escrito (qué buscaba cada adición).
- Registrar temperatura real y tiempo real de whirlpool.
- Definir método de separación y estimación de pérdidas.

## 3) Modo diagnóstico (rápido)
Si el lote sale mal, ubicalo en uno de estos 3 buckets:

A) “Falta aroma / aroma muerto” → mirar oxígeno primero, luego timing (IPA Parte 2).  
B) “Rasposo / hop burn / té” → mirar whirlpool (temp/tiempo) + polifenoles + dry hop.  
C) “Amargor duro” → mirar agua/pH + hot-side + oxidación.
