# Whirlpool: temperatura y tiempo

## 1) Por qué el whirlpool es el corazón del hot-side moderno
El whirlpool permite extraer y fijar perfil aromático con menos destrucción que el hervor, pero todavía con:
- extracción relevante
- posibilidad de generar amargor si la temperatura queda alta demasiado tiempo
- riesgo de “té/rasposo” si sobre-extraés

## 2) Las 3 variables críticas (en orden)
### A) Temperatura real
No la del setpoint: la real en el mosto.
- Más alta: más extracción + más riesgo de amargor/aspereza
- Más baja: menos extracción “verde”, mejor control del amargor suave

### B) Tiempo real de contacto
- Más tiempo no siempre significa más aroma “útil”.
- Más tiempo sí suele significar más extracción de materia (polifenoles/vegetal).

### C) Agitación / recirculación
- Aumenta transferencia, pero también aumenta riesgo de sobre-extracción y arrastre.

## 3) Estrategia robusta (baseline)
- Definir una temperatura objetivo de whirlpool y sostenerla (o definir la caída).
- Limitar el tiempo de contacto (y documentarlo).
- Definir cómo se enfría y cómo se separa el material.

## 4) Señales de diagnóstico
- Si la IPA sale “áspera” sin ser muy amarga: revisá temp/tiempo de whirlpool.
- Si sale aromáticamente “plana” pese a dry hop fuerte: revisá que el hot-side esté aportando “base” (no solo IBU).
