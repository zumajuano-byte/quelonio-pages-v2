# Estabilidad vs expresividad

## 1) El tradeoff real
En IPA moderna, subir expresividad (más carga, más extracción, más turbidez) suele aumentar:
- riesgo de hop burn/astringencia
- pérdidas por absorción
- sensibilidad a oxígeno
- inestabilidad (shelf-life más corta)

La decisión correcta depende del “producto”:
- ¿cerveza para consumo inmediato (taproom) o distribución?
- ¿cadena de frío real o variable?

## 2) Marco de decisión (simple)
Elegí el “perfil objetivo” y alineá proceso:

### A) Máxima expresividad (taproom / fresco)
- Mayor tolerancia a inestabilidad
- Requiere disciplina de oxígeno igual (si no, muere en días)
- Control fuerte de hop burn y separación

### B) Expresividad balanceada (distribución corta)
- Diseñar dry hop para eficiencia (no saturación)
- DO como variable no negociable
- Control de color y estabilidad sensorial

### C) Estabilidad prioritaria (shelf-life más largo)
- Reducir carga vegetal y contacto
- Subir foco en whirlpool “limpio”
- Envasado y control de oxígeno al máximo

## 3) Checklist de cierre (para operar)
- [ ] Definiste tu objetivo de vida útil (días/semanas en tu cadena).
- [ ] Ajustaste el esquema de dry hop para ese objetivo.
- [ ] Tenés estándar de purga/transferencia/envasado.
- [ ] Tenés log de lupulado (para repetir y diagnosticar).

## Marco de decisión — Estabilidad vs Expresividad (operativo)

### Objetivo
Tomar decisiones de diseño sin “romanticismo”: cada punto de expresividad tiene costo en estabilidad.

### Matriz (rápida)
| Decisión | Sube expresividad | Sube riesgo | Cómo compensar |
|---|---|---|---|
| Dosis DH muy alta | Sí | Hop burn / creep / oxidación | Fraccionar, limitar contacto, controlar temp |
| Contacto largo | A veces | Polifenoles / vegetal / burn | Tiempo máximo, mejor separación |
| Dry hop tardío | Sí | Oxidación | Circuito cerrado + purgas + minimizar headspace |
| Mucho haze buscado | Puede | Shelf-life menor / textura inestable | Control de proceso + frío + rotación |

### Checklist de diseño (antes de producir)
- [ ] ¿Cuál es el “target” real? (NEIPA jugosa vs WC nítida vs híbrida)
- [ ] ¿Cuál es la ventana de consumo? (rápida / media / larga)
- [ ] ¿Cuál es el riesgo principal del lote? (O₂ / creep / burn / color)
- [ ] ¿Qué control “no negocio” sí o sí? (purga, tiempos, temperaturas, registros)

### Regla interna (útil)
- Si el producto necesita venderse “fresco”, priorizar expresividad.
- Si el producto viaja o rota lento, priorizar estabilidad.
