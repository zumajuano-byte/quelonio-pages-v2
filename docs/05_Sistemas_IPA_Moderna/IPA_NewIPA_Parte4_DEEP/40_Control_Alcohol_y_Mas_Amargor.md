# Control de alcohol y “más amargor” sin romper el sistema

## 1) Alcohol (ABV) como variable de diseño
Cambiar ABV suele cambiar:
- percepción de amargor (más alcohol puede endurecer)
- cuerpo y sequedad percibida
- “soporte” para aroma

Regla operativa: si movés ABV, revisá balance (agua + cuerpo + amargor base).

## 2) “Quiero más amargor” (la trampa típica)
En IPA moderna, subir amargor “a lo bruto” suele disparar:
- aspereza (whirlpool muy caliente/largo)
- amargor duro con el tiempo (oxidación)
- choque con haze/textura

## 3) Ajuste correcto por capas
- Falta amargor: ajustar **capa base** (controlada) sin tocar dry hop.
- Amargor duro: corregir hot-side (whirlpool) y agua/pH antes de bajar aroma.
- Amargor “hueco”: revisar soporte (cuerpo) y/o aroma (nariz sostiene percepción).
