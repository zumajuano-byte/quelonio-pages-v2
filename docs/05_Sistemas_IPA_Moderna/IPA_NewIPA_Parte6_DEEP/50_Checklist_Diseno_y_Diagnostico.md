# Checklist — Diseño y diagnóstico (IPA moderna)

Este checklist es el “gate” mínimo para salir a producción y para diagnosticar sin caos.

---

## A) Diseño (antes de producir)
- [ ] Objetivo sensorial en 1 frase (nota líder + 2 secundarias).
- [ ] Definida arquitectura por capas:
  - [ ] Amargor base (capa A)
  - [ ] Hot-side/whirlpool (capa B)
  - [ ] Dry hop (capa C)
- [ ] Definido si se busca biotransformación (sí/no) y cómo.
- [ ] Receta como plataforma: cuerpo/fermentabilidad definidos (mash y malta).
- [ ] Agua: intención SO4/Cl y pH mash objetivo.
- [ ] Plan de oxígeno: dry hop, transferencias y envasado.

---

## B) Ejecución (durante el lote)
- [ ] Registrada temperatura/tiempo real de whirlpool.
- [ ] Registrado timing de dry hop (día/etapa/temperatura/horas).
- [ ] Registradas cargas (g/L sobre volumen real) y variedades/lotes.
- [ ] Gate de diacetilo/VDK considerado antes de enfriar (si dry hop alto).
- [ ] Transferencias cerradas y purgas aplicadas según estándar.

Logs:
- TP Lupulado: `../../04_Lupulo/Lupulo_Parte5_DEEP/90_TP_Log_Lupulado.md`
- Plantilla canónica: `10_Plantilla_Canonica_IPA_Moderna.md`

---

## C) Validación (post-lote)
- [ ] Aroma: intensidad + descriptor (nota líder).
- [ ] Amargor: calidad (suave vs duro).
- [ ] Hop burn / astringencia: presente/ausente y severidad.
- [ ] Color: estable o tendencia a oscurecer.
- [ ] Estabilidad: ¿cuántos días aguanta la expresión aromática en tu cadena?

---

## D) Ajuste (solo 1 palanca por lote)
- [ ] Si falta amargor: tocar capa A (base), no dry hop.
- [ ] Si falta aroma: revisar oxígeno primero, luego timing y método.
- [ ] Si hay rasposo/hop burn: tocar contacto/temperatura/separación antes de bajar aroma.
- [ ] Si está “thin/afilada”: tocar soporte (mash/malta) y agua antes de subir lúpulo.
