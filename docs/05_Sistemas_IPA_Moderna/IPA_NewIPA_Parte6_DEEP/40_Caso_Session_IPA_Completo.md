# Caso Session IPA completo

## 1) Objetivo sensorial (ejemplo)
- “Muy aromática, liviana, final seco, amargor medio-suave, alta tomabilidad, estabilidad razonable.”

## 2) Desafío estructural
Con menos alcohol y menos cuerpo “natural”, la Session tiende a quedar:
- filosa / delgada si no hay soporte
- aromáticamente intensa pero corta si hay oxidación o mala plataforma

## 3) Arquitectura recomendada
- Amargor base moderado y limpio.
- Whirlpool eficiente (no exagerado) para crear base aromática.
- Dry hop diseñado para impacto (variedades con alto rendimiento sensorial) evitando saturación vegetal.

## 4) Receta como plataforma (principios)
- Construir soporte desde mash (cuerpo sin dulzor).
- Ajustar agua para evitar filo excesivo.
- Carbonatación y servicio importan más en percepción final (session “muestra” defectos).

## 5) Riesgos típicos y controles
- “Thin + harsh”: corregir soporte y agua antes de subir lúpulo.
- Hop burn: contacto/temperatura/separación.
- Oxidación: en session se nota rápido; proceso cerrado.
