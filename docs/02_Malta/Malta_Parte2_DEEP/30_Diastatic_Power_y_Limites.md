# Diastatic Power (DP) y límites

## 1) Qué es DP (en términos útiles)
DP (Diastatic Power) es una medida de la capacidad de la malta para **convertir almidón en azúcar**, y por eso es un indicador directo de “aptitud para macerar” cuando la receta exige conversión.

La misma fuente lo define de forma simple: DP refiere a la capacidad de convertir almidón a azúcar y es esencial para brewing.

## 2) Dónde te pega: adjuntos y maltas especiales
Cuando metés adjuntos (o fuentes no malteadas) estás agregando material que puede aportar extracto, pero **te exige pensar en actividad enzimática disponible** para degradar/convertir.

## 3) Regla operativa (sin “números mágicos”)
En vez de fijar umbrales universales (que varían por maltster y sistema), usá este marco:

### Si la receta tiene alta proporción de material con bajo DP
- Asegurar que la porción de malta base tenga “reserva” enzimática suficiente
- Evitar perfiles térmicos que perjudiquen el objetivo de conversión
- Registrar conversión real (tiempo hasta conversión + densidad/eficiencia del sistema)

### Si el objetivo es consistencia
- Cerrar el loop con COA (cuando esté disponible): DP + métricas de modificación + humedad
- Ajustar tiempo/temperatura por lote (no por “costumbre”)

## 4) Checklist de decisión
- [ ] % adjuntos / maltas con bajo DP
- [ ] Objetivo de FG / atenuación (seco vs cuerpo)
- [ ] Tiempo disponible en sala de cocción (tradeoff real)
- [ ] Perfil térmico (ver sección 40)

## Reglas (B004)

<a id="r-0086"></a>
### R-0086 — Usar Diastatic Power (DP) como criterio de “capacidad de conversión” del grist

- FuenteID: B004
- Evidencia: "diastatic power... a measure of a malt's ability to convert starch to sugar"
- Ubicación: B004 p38

**Regla (operativa):**
- Si el grist incluye proporciones altas de insumos con baja actividad enzimática, asegurar que el total del grist conserve capacidad de conversión (DP suficiente en las bases).
- Ante problemas de conversión, revisar DP/enzimas antes de “corregir” con azúcar o cambios agresivos.

**Aplicación / límites:**
- Aplica especialmente en grists con adjuntos o maltas con bajo poder diastático.
- B004 aquí define concepto; números objetivo dependen de tu estándar interno/fuente complementaria.
