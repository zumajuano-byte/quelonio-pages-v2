# Matriz de decisión (Malta) — “qué mover primero”

Usá esta matriz para decidir qué palanca tocar primero sin entrar en prueba y error infinito.

## 1) Matriz (síntoma → palanca → control)
| Síntoma / objetivo | Palanca 1 (primaria) | Palanca 2 | Qué medir/registrar |
|---|---|---|---|
| Falta cuerpo / “thin” | mash (perfil térmico) | % especiales (con criterio) | pH, T°, tiempo, FG |
| Mucho dulzor / pesado | mash (más fermentabilidad) | bajar caramelo / ajustar agua | FG, balance SO4/Cl, pH |
| Áspero/astringente | sparge/pH + roasted | bajar sulfatos / revisar molienda | pH mash+sparing, sales, grist |
| Color muy oscuro | % especiales/roast | hervor (tiempo/energía) | color objetivo vs real |
| Espuma pobre | receta (proteína) + proceso | limpieza + fermentación | sensorial, proceso, higiene |
| Inconsistencia lote a lote | COA/lote + molienda | pH control y registro | TP malta + TP agua |

## 2) Reglas simples (anti-caos)
- Cambiá 1 variable por vez (cuando estés “diagnosticando”).
- Registrá siempre: pH mash, T°, tiempo, % especiales, y lote de malta.
- Si cambiás roasted/caramelo, revalidá agua/pH.
