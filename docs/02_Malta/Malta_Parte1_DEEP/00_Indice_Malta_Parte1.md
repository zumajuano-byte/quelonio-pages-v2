# Malta — Parte 1 (Fundamentos operables)

Esta parte establece lo mínimo no negociable para que Malta “acomode” el proceso:
- entender el malteado (steeping → germinación → kilning)
- entender DP/enzimas (qué convierte y cómo afecta cuerpo/fermentabilidad)
- entender reacciones de pardeamiento (Maillard/caramelo/tostado)
- operar con COA (predecir performance)
- moler bien (evitar stuck / polvo / granos enteros)
