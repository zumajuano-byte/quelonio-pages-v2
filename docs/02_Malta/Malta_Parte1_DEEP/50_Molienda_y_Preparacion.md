# Molienda y preparación (donde se arruinan muchos lotes)

## 1) Objetivo real
- exponer endospermo para conversión
- mantener cáscara útil para lautering

---

## 2) Dos fallas típicas (operativas)
- exceso de polvo → compactación, lautering lento, stuck
- granos enteros → conversión incompleta, baja eficiencia

Tu resumen lo marca como riesgo típico y distingue seca vs húmeda.

---

## 3) SOP mínimo
1) Definir objetivo (receta / % especiales / riesgo de stuck).
2) Ajustar gap/setting.
3) Validar con inspección visual (cáscara / endospermo / polvo).
4) Registrar setting y resultado.
