# Molienda y preparación (donde se arruinan muchos lotes)

## 1) Objetivo real
- exponer endospermo para conversión
- mantener cáscara útil para lautering

---

## 2) Dos fallas típicas (operativas)
- exceso de polvo → compactación, lautering lento, stuck
- granos enteros → conversión incompleta, baja eficiencia

Tu resumen lo marca como riesgo típico y distingue seca vs húmeda.

---

## 3) SOP mínimo
1) Definir objetivo (receta / % especiales / riesgo de stuck).
2) Ajustar gap/setting.
3) Validar con inspección visual (cáscara / endospermo / polvo).
4) Registrar setting y resultado.

## Reglas (B004)

<a id="r-0070"></a>
### R-0070 — Definir el objetivo de molienda: exponer endospermo con mínima destrucción de cáscara

- FuenteID: B004
- Evidencia: "breaking or cracking the husk of dried malt to expose the starchy endosperm"
- Ubicación: B004 p66

**Regla (operativa):**
- Ajustar molienda para exponer endospermo sin destruir cáscara más de lo necesario.
- Documentar setting (gap/rodillos) cuando cambie el lauter o el extracto.

**Aplicación / límites:**
- Aplica siempre que cambie molino, rodillos, proveedor de malta o condición de grano.
- No define granulometría objetivo (en este extracto no hay tamices/valores).

<a id="r-0071"></a>
### R-0071 — Evitar extremos: over-milling vs under-milling tienen fallas distintas (flujo vs extracto)

- FuenteID: B004
- Evidencia: "Over-milling can lead to excessive dust... while under-milling can leave uncrushed kernels, resulting in low extract yields."
- Ubicación: B004 p67

**Regla (operativa):**
- Si hay problemas de lauter/flujo: sospechar over-milling (polvo) como causa candidata.
- Si baja el extracto/eficiencia: sospechar under-milling (kernels sin triturar) como causa candidata.

**Aplicación / límites:**
- Aplica para diagnóstico rápido de performance.
- No reemplaza grist analysis formal (B004 menciona “grist analysis” pero el detalle no está en este extracto).

<a id="r-0072"></a>
### R-0072 — Considerar wet milling cuando el polvo sea un riesgo o limitante (y para mejorar husk/porosidad)

- FuenteID: B004
- Evidencia: "Wet milling involves adding water to grain before crushing, improving husk pliability and reducing dust."
- Ubicación: B004 p67

**Regla (operativa):**
- Si el polvo es problema (limpieza, operación, seguridad), evaluar wet milling como alternativa operativa.
- Tener en cuenta que wet milling requiere gestión cuidadosa de interacción agua-grano (operación más sensible).

**Aplicación / límites:**
- Aplica cuando tu sistema/escala y equipo lo permitan.
- No implica que “siempre es mejor”: depende de equipamiento y control.

<a id="r-0073"></a>
### R-0073 — Diseñar/usar silos/bins pensando en flujo y limpieza del sistema (calidad + seguridad)

- FuenteID: B004
- Evidencia: "Silos and bins used for storing malt must facilitate easy removal... minimizing hang-ups."
- Ubicación: B004 p65

**Regla (operativa):**
- Almacenar malta con criterio de flujo (evitar “hang-ups”) y con posibilidad real de limpieza del sistema.
- Tratar handling/limpieza como parte de “calidad de malta” (no solo logística).

**Aplicación / límites:**
- Aplica si manejás bins/silos o tolvas (aunque sean chicas).
- Si usás bolsas/bolsones: aplicar el espíritu (flujo/limpieza), aunque el hardware sea distinto.
