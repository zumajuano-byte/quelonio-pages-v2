# Química de malta para cerveceros (DP + enzimas + resultado)

## 1) DP (Diastatic Power): capacidad de conversión
DP es la capacidad de la malta de convertir almidón en azúcares fermentables.

Traducción práctica:
- si DP es bajo y la receta depende de conversión (adjuntos, % alto de especiales), podés tener conversión lenta/incompleta.

---

## 2) Enzimas clave (idea operativa)
- α-amilasa y β-amilasa son críticas para convertir almidón y definir perfil de azúcares (cuerpo vs fermentabilidad).

Traducción práctica:
- no es solo “convertir”: es **qué mix de azúcares** se genera, y eso se siente en cuerpo/dulzor/atenuación.

---

## 3) Guardrails mínimos
- Si la conversión se vuelve errática: mirar primero **molienda**, después **pH del mash (Agua)**, después **DP/receta**.
- Puente directo a Agua: [01 — Agua](../../01_Agua/01_Agua.md)

---

## 4) Qué registrar (mínimo)
- tipo de malta (base/especial/tostada)
- % en receta
- DP (si COA lo trae) y observaciones de conversión
- resultado: conversión / lautering / eficiencia (si medís)

## Reglas (B004)

<a id="r-0066"></a>
### R-0066 — Considerar beta glucanos/hemicelulosa como “palancas” de filtración (vía malteado)

- FuenteID: B004
- Evidencia: "These polysaccharides, which include beta glucans and hemicellulose, are crucial for cell wall structure and are degraded during the malting process"
- Ubicación: B004 p36–p37

**Regla (operativa):**
- Si hay problemas de filtración/flujo, evaluar si el “grado de degradación” esperado por malteado puede ser parte del problema (no asumir que es solo molienda).
- Tratar beta glucanos/NSP como tema a revisar en COA/especificación y en comunicación con maltster.

**Aplicación / límites:**
- Aplica cuando aparece variación de filtración o lautering entre lotes/proveedores.
- No define valores objetivo (B004 no da números en este extracto).
- Soporte: NSP y degradación durante malteado.

<a id="r-0067"></a>
### R-0067 — Usar “Diastatic Power” como criterio de aptitud de conversión

- FuenteID: B004
- Evidencia: "a measure of a malt's ability to convert starch to sugar"
- Ubicación: B004 p38

**Regla (operativa):**
- Tratar DP como señal de “capacidad de conversión” (especialmente relevante cuando el grist incluye adjuntos o maltas con baja actividad).
- Documentar el DP del lote cuando haya cambios de eficiencia o conversión.

**Aplicación / límites:**
- Aplica en diseño de grist y diagnóstico de conversión.
- No define rangos/umbrales (en este extracto no hay límites numéricos).
- Soporte: definición de DP.

<a id="r-0076"></a>
### R-0076 — Tratar el kernel como “fábrica”: transforma carbohidratos, proteínas y lípidos vía acción enzimática

- FuenteID: B004
- Evidencia: "The barley kernel acts as a miniature factory, transforming carbohydrates, proteins, and lipids"
- Ubicación: B004 p35

**Regla (operativa):**
- Cuando cambie cuerpo/espuma/estabilidad, no pensar solo en “azúcares”: revisar también proteínas y lípidos (vía malteado + manejo).
- Usar esta regla como checklist conceptual antes de tocar receta.

**Aplicación / límites:**
- Aplica en diagnóstico sensorial/estabilidad y consistencia lote a lote.
- No define valores objetivo; solo ordena qué variables mirar.

<a id="r-0077"></a>
### R-0077 — La modificación en germinación es secuencial: el endospermo se degrada “de un extremo al otro”

- FuenteID: B004
- Evidencia: "This enzymatic activity is sequential, progressing from one end of the endosperm to the other"
- Ubicación: B004 p35

**Regla (operativa):**
- Interpretar “modificación” como proceso direccional y progresivo (útil para entender variaciones de conversión/filtración).
- Si hay señales de sub-modificación, priorizar revisar COA/atributos de modificación antes de cambiar molienda.

**Aplicación / límites:**
- Aplica en diagnóstico de conversión y lautering.
- No reemplaza ensayos; guía el orden de revisión.
