# REGISTRO_FUENTES — Catálogo de bibliografía (FuenteID)

Este registro define las **fuentes válidas** para reglas en la Biblia.
Una regla solo es válida si cita: `FuenteID + ubicación exacta + cita textual corta`.

## Convenciones
- FuenteID: `B###` (ej: B022)
- Ubicación: `p##`, `fig## p##`, `tbl## p##`
- Estado: `PENDIENTE / EN PROCESO / CERRADO`
- Idioma: `ES / EN / MIXTO / OCR`
