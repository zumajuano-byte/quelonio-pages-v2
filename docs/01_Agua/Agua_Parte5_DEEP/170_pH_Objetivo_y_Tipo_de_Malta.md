# pH objetivo según tipo de malta

## Maltas base claras
- Tendencia a subir pH
- Requieren alcalinidad baja o nula
- Objetivo típico: pH 5,3–5,4

## Maltas oscuras / tostadas
- Aportan acidez natural
- Pueden tolerar alcalinidad mayor
- Objetivo típico: pH 5,4–5,6

## Regla operativa
Nunca definir pH objetivo sin evaluar:
- proporción de maltas especiales
- color estimado (SRM)
