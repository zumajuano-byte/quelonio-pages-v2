# Estrategias de ajuste del mash

## Opciones disponibles
1. Ajuste por sales cálcicas
2. Ajuste por ácidos (láctico / fosfórico)
3. Uso de RO + blending
4. Ajuste vía selección de malta

## Prioridad recomendada
1. Diseñar alcalinidad
2. Ajustar calcio
3. Corregir pH fino con ácido

Siempre validar con medición real.
