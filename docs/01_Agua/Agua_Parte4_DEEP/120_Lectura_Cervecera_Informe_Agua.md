# Lectura cervecera de un informe de agua

Este capítulo traduce un informe químico municipal/laboratorio a decisiones cerveceras reales.

## Parámetros realmente críticos
- Calcio (Ca²⁺)
- Magnesio (Mg²⁺)
- Alcalinidad total (como CaCO₃)
- Cloruros (Cl⁻)
- Sulfatos (SO₄²⁻)
- Sodio (Na⁺)
- Desinfectantes (cloro / cloraminas)

## Parámetros secundarios
Hierro, manganeso, nitratos: relevantes solo si superan umbrales sensoriales.

## Error estructural común
Leer el pH del agua como indicador directo del mash. El pH del agua **no predice** el pH del mash.
