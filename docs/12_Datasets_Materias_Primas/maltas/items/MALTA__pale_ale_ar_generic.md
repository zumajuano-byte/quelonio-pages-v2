# MALTA__pale_ale_ar_generic

## Descripción
Malta base pale ale genérica del mercado AR, equivalente a maltas pale ale europeas como Maris Otter o importadas similares.

## Región
AR (Argentina-first).

## Notas de importación
No aplica.

## Incidencia por estilo
Ver “Incidencia estándar”: ../../_schema/INCIDENCIA_ESTANDAR.md

Regla: Si hay datos Excel/Sheets, usar incidencia real; sino, heurística.

## Parámetros clave
- Color (EBC): 3-5.
- Extracto potencial: 80-83%.
- Humedad: 4-5%.
- Proteína: 10-12%.
- Poder diastático: 200-300 WK.
- Notas sensoriales: Neutra, con toques de pan tostado leve.

### Uso típico
Estilos ale (pale ale, IPA, NEIPA); rango de uso: 80-100%; límites/alertas: Combinar con maltas especiales para evitar perfil plano.
