# LUPULO__citra_generic

## Descripción
Lúpulo Citra genérico, teórico para dry hopping en IPAs modernas.

## Región
INT (Importado; Argentina no produce Citra localmente).

## Notas de importación
Costos elevados por flete; trazabilidad crítica para lotes.

Ver Incidencia estándar: [_schema/INCIDENCIA_ESTANDAR.md](../../_schema/INCIDENCIA_ESTANDAR.md)

## Parámetros clave
- Alfa ácidos: 12-14%.
- Aceites: Alto en cítricos.
- Uso típico: Aroma en NEIPAs.
