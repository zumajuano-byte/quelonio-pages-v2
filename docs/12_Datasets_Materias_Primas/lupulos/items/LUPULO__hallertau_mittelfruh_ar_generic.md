# LUPULO__hallertau_mittelfruh_ar_generic

## Descripción
Lúpulo Hallertau Mittelfrüh genérico del mercado AR, equivalente a variedades Hallertau Mittelfrüh alemanas importadas similares.

## Región
AR (Argentina-first).

## Notas de importación
No aplica (Argentina-first); variabilidad de lote común; importados suelen tener info más consistente del distribuidor.

## Incidencia por estilo
Ver “Incidencia estándar”: ../../_schema/INCIDENCIA_ESTANDAR.md

Regla: Si hay datos Excel/Sheets, usar incidencia real; sino, heurística.

## Parámetros clave
- Alfa ácidos: 3-5%.
- Beta ácidos: 3-4%.
- Aceites totales: 0.5-1.5%.
- Notas sensoriales: Noble, floral, hierba fresca.

### Lo confiable
Alfa ácidos idealmente confirmar por COA/proveedor; si no hay COA, usar rango Biblia como REFERENCIA.

### Uso típico
Estilos lager, Weissbier; objetivo: aroma noble y balance; dosificación orientativa: 15-30 IBU para amargor (hervor 60 min), 10-20 min para whirlpool/aroma (5-10 g/L), dry hop 3-5 días (5-10 g/L).

## Referencias
- COA/Excel manda sobre Biblia. Agua no se asume.
