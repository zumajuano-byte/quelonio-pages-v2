# LUPULO__columbus_ctz_ar_generic

## Descripción
Lúpulo Columbus/CTZ genérico del mercado AR, equivalente a variedades Columbus/CTZ importadas similares.

## Región
AR (Argentina-first).

## Notas de importación
No aplica (Argentina-first); variabilidad de lote común; importados suelen tener info más consistente del distribuidor.

## Incidencia por estilo
Ver “Incidencia estándar”: ../../_schema/INCIDENCIA_ESTANDAR.md

Regla: Si hay datos Excel/Sheets, usar incidencia real; sino, heurística.

## Parámetros clave
- Alfa ácidos: 14-18%.
- Beta ácidos: 4-5.5%.
- Aceites totales: 2-4%.
- Notas sensoriales: Cítrico intenso, pino, hierba.

### Lo confiable
Alfa ácidos idealmente confirmar por COA/proveedor; si no hay COA, usar rango Biblia como REFERENCIA.

### Uso típico
Estilos IPA, NEIPA; objetivo: amargor alto y aroma cítrico; dosificación orientativa: 30-60 IBU para amargor (hervor 60 min), 10-20 min para whirlpool (10-20 g/L), dry hop 3-7 días (10-20 g/L).

## Referencias
- COA/Excel manda sobre Biblia. Agua no se asume.
