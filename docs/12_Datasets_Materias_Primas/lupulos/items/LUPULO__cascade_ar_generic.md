# LUPULO__cascade_ar_generic

## Descripción
Lúpulo Cascade genérico del mercado AR, equivalente a variedades Cascade importadas similares.

## Región
AR (Argentina-first).

## Notas de importación
No aplica (Argentina-first); variabilidad de lote común; importados suelen tener info más consistente del distribuidor.

## Incidencia por estilo
Ver “Incidencia estándar”: ../../_schema/INCIDENCIA_ESTANDAR.md

Regla: Si hay datos Excel/Sheets, usar incidencia real; sino, heurística.

## Parámetros clave
- Alfa ácidos: 4-7%.
- Beta ácidos: 3-4.5%.
- Aceites totales: 1-2.5%.
- Notas sensoriales: Floral, cítrico, pino.

### Lo confiable
Alfa ácidos idealmente confirmar por COA/proveedor; si no hay COA, usar rango Biblia como REFERENCIA.

### Uso típico
Estilos IPA, APA; objetivo: amargor y aroma; dosificación orientativa: 20-40 IBU para amargor (hervor 60 min), 5-15 min para whirlpool/aroma (10-20 g/L), dry hop 5-10 días (5-15 g/L).

## Referencias
- COA/Excel manda sobre Biblia. Agua no se asume.
