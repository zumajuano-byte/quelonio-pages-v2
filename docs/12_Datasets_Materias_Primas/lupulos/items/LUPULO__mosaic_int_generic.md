# LUPULO__mosaic_int_generic

## Descripción
Lúpulo Mosaic importado típico, equivalente a variedades Mosaic de EE.UU. o similares.

## Región
INT (importado típico).

## Notas de importación
Importado; info consistente del distribuidor; sin afirmar disponibilidad.

## Incidencia por estilo
Ver “Incidencia estándar”: ../../_schema/INCIDENCIA_ESTANDAR.md

Regla: Si hay datos Excel/Sheets, usar incidencia real; sino, heurística.

## Parámetros clave
- Alfa ácidos: 10-14%.
- Beta ácidos: 3-4.5%.
- Aceites totales: 1.5-3%.
- Notas sensoriales: Frutal (mora, blueberry), cítrico tropical.

### Lo confiable
Alfa ácidos idealmente confirmar por COA/proveedor; si no hay COA, usar rango Biblia como REFERENCIA.

### Uso típico
Estilos NEIPA, IPA moderna; objetivo: aroma frutal y complejidad; dosificación orientativa: 25-50 IBU para amargor (hervor 60 min), 10-20 min para whirlpool (10-20 g/L), dry hop 4-8 días (10-20 g/L).

## Referencias
- COA/Excel manda sobre Biblia. Agua no se asume.
