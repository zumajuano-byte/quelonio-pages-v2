# LEVADURA__s04_market

## Descripción
Levadura seca S-04 genérica de mercado, perfil inglés para ales tradicionales.

## Región
INT (importada, disponible en proveedores locales).

## Notas de importación
Fácil disponibilidad; verificar lote y fecha de caducidad.

## Incidencia por estilo
Ver “Incidencia estándar”: ../../_schema/INCIDENCIA_ESTANDAR.md

Regla: Si hay datos Excel/Sheets, usar incidencia real; sino, heurística.

## Parámetros clave
- Atenuación: Media (68-75%).
- Floculación: Media-alta.
- Tolerancia alcohol: Media (hasta 8-10% ABV).
- Temperatura típica de fermentación: 18-22°C.

### Recomendación de rehidratación / pitch
Rehidratar en agua estéril tibia; pitch rate moderada para complejidad; fermentar en rango para balance.

### Riesgos comunes
Estres por temperatura alta (esteres excesivos); subpitch (fermentación incompleta); diacetilo si rampas rápidas.
