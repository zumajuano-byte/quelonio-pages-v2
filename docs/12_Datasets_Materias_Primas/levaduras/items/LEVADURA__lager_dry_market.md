# LEVADURA__lager_dry_market

## Descripción
Levadura seca lager genérica de mercado, perfil neutral para lagers claras.

## Región
INT (importada, disponible en proveedores locales).

## Notas de importación
Fácil disponibilidad; verificar lote y fecha de caducidad.

## Incidencia por estilo
Ver “Incidencia estándar”: ../../_schema/INCIDENCIA_ESTANDAR.md

Regla: Si hay datos Excel/Sheets, usar incidencia real; sino, heurística.

## Parámetros clave
- Atenuación: Alta (75-82%).
- Floculación: Media-alta.
- Tolerancia alcohol: Alta (hasta 10-12% ABV).
- Temperatura típica de fermentación: 10-15°C (con rampas controladas).

### Recomendación de rehidratación / pitch
Rehidratar en agua estéril fría; pitch rate alta para claridad; fermentar frío con rampas lentas para limpieza.

### Riesgos comunes
Estres por temperatura alta (diacetilo, esteres); subpitch (fermentación lenta); contaminación si frío insuficiente.
