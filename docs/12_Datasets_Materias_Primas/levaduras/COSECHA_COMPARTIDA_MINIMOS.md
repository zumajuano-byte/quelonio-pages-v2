# Checklist Mínimos para Cosecha Compartida de Levadura

## Introducción
Cuando una cervecería comparte cosecha de levadura, registrar mínimos para reducir riesgos. Si no hay datos, tratar como riesgo alto.

## Checklist de Datos Mínimos a Registrar
- Fecha de cosecha y generación (e.g., primera, segunda repitch).
- Densidad celular (e.g., viable cells/mL).
- Temperatura de fermentación y maduración.
- Tiempo total de fermentación.
- Apariencia (color, turbidez).
- Aroma (notas limpias o defectos).
- Método de sanitización (e.g., alcohol, peróxido).
- Condiciones de transporte (temperatura, tiempo).

## Recomendaciones
- Si no hay al menos estos mínimos, rechazar o tratar como riesgo alto.
- Ideal: Usar microscopio y/o test simple de viabilidad (e.g., viabilidad >80%).
- Análisis profundo (COA completo) se hará más adelante en módulo laboratorio.

## Notas
- Siempre priorizar COA/Excel sobre supuestos. Agua no se asume.
