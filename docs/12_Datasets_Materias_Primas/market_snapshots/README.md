# Market Snapshots

## Qué va
- Referencias temporales de precios (rangos, índices, niveles).
- Tendencias de disponibilidad.
- Fuentes locales primero (Argentina-first).

## Qué no va
- Precios exactos o duros.
- Datos sensibles o no verificables.
- Sin fuente o fecha clara.

## Naming sugerido
- [MATERIAL]_[FUENTE]_[FECHA].md
- Ejemplo: MALTA_PILSEN_AR_LOCAL_2025-01.md
