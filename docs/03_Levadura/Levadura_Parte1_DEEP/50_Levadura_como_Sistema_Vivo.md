# La levadura como sistema vivo

La levadura:
- evoluciona
- se adapta
- se fatiga
- muta

No es un insumo inerte.
Reutilizar levadura implica **gestión biológica**, no solo ahorro de costos.

## Reglas núcleo (Parte 1)

<a id="r-0096"></a>
### R-0096 — Registrar y gestionar levadura como sistema vivo: desempeño cambia con cepa y generaciones

- FuenteID: B003
- Evidencia: "Diferentes cepas (y generaciones) exhiben atributos únicos y dan diferentes resultados."
- Ubicación: B003 p25 (PDF pág. 25)

**Regla (operativa):**
- Llevar log mínimo por lote: cepa, generación, condición, desempeño (tiempo, atenuación, off-flavors).
- No asumir que “la misma levadura” se comporta igual sin control de generación/estado.

**Aplicación / límites:**
- Aplica siempre; crítico si reusás levadura.
- Si no registrás, no podés diagnosticar ni repetir.
