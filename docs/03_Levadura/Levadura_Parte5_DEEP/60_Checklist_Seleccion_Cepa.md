# Checklist — Selección de cepa

Este checklist cierra la decisión: si no podés tildarlo, **no está cerrada la elección**. :contentReference[oaicite:15]{index=15}

---

## A) Objetivo y restricción (inputs)

- [ ] **Perfil sensorial definido** (limpio / ésteres / fenoles / mixto) :contentReference[oaicite:16]{index=16}
- [ ] Estilo objetivo (solo como marco, no como “jaula”)
- [ ] OG / alcohol objetivo definido
- [ ] **Temperatura disponible real** (mín/máx sostenibles) :contentReference[oaicite:17]{index=17}
- [ ] Limitaciones operativas claras (capacidad de frío, tiempos, oxígeno disponible, etc.)

---

## B) Selección por comportamiento (decisión)

- [ ] Categoría de carácter elegida (limpia / frutada / híbrida / fenólica) :contentReference[oaicite:18]{index=18}
- [ ] Floculación objetivo definida (baja/media/alta) y método para lograr claridad :contentReference[oaicite:19]{index=19}
- [ ] Trade-offs explicitados (velocidad vs control; floculación vs limpieza; etc.) :contentReference[oaicite:20]{index=20}

---

## C) Plan de fermentación (salida obligatoria)

- [ ] **Pitch rate definido** :contentReference[oaicite:21]{index=21}
- [ ] Oxigenación definida (método + timing)
- [ ] Curva térmica definida (setpoints y rampas)
- [ ] Nutrientes definidos (si aplica)
- [ ] **Plan de fermentación completo** (incluye plan de limpieza/diacetilo) :contentReference[oaicite:22]{index=22}

---

## D) Repitch / preservación de cepa (si aplica)

- [ ] ¿Se reutiliza? (sí/no) + generaciones objetivo
- [ ] Plan de cosecha y almacenamiento definido
- [ ] Higiene: limpieza + sanitización entre batches definida :contentReference[oaicite:23]{index=23}

## Reglas operativas (B003) — Checklist

### R-0133 — En cono: cosecha con criterio y descartes para evitar trub/deriva {#r-0133}
**Regla:** si cosechás de cono, evita el “fondo sucio” y estandarizá descartes.  
**Evidencia (B003):** “desecha el primer tercio… contiene más trub… células muertas…”. :contentReference[oaicite:24]{index=24}

### R-0134 — Si hay manejo de cultivo/propagación: la selección limpia es obligatoria {#r-0134}
**Regla:** si vas a cultivar/propagar, priorizá evitar contaminantes y mantener consistencia de cepa.  
**Evidencia (B003):** “La selección adecuada de colonias es una parte vital…”. :contentReference[oaicite:25]{index=25}

### R-0135 — La decisión de cepa debe incluir setpoints coherentes (no sólo “la cepa”) {#r-0135}
**Regla:** la salida mínima de “elegí cepa” es: pitch/O2/temperatura/nutrientes coherentes con el objetivo.  
**Evidencia (Parte 5):** “Qué setpoints son coherentes con esa elección (pitch/O2/temperatura/nutrientes)”. :contentReference[oaicite:26]{index=26}

### R-0136 — La decisión se valida con caso + checklist antes de ejecutar {#r-0136}
**Regla:** cerrás la elección cuando pasa checklist y aplica en un caso (Ale Americana / Lager / NEIPA).  
**Evidencia (Parte 5):** “Cómo validar la elección con un caso aplicado y un checklist”. :contentReference[oaicite:27]{index=27}
