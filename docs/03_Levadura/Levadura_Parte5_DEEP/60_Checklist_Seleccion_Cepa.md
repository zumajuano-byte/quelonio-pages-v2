# Checklist — Selección de cepa

Este checklist cierra la decisión: si no podés tildarlo, **no está cerrada la elección**.

---

## A) Objetivo y restricción (inputs)

- [ ] **Perfil sensorial definido** (limpio / ésteres / fenoles / mixto)
- [ ] Estilo objetivo (solo como marco, no como “jaula”)
- [ ] OG / alcohol objetivo definido
- [ ] **Temperatura disponible real** (mín/máx sostenibles)
- [ ] Limitaciones operativas claras (capacidad de frío, tiempos, oxígeno disponible, etc.)

---

## B) Selección por comportamiento (decisión)

- [ ] Categoría de carácter elegida (limpia / frutada / híbrida / fenólica)
- [ ] Floculación objetivo definida (baja/media/alta) y método para lograr claridad
- [ ] Trade-offs explicitados (velocidad vs control; floculación vs limpieza; etc.)

---

## C) Plan de fermentación (salida obligatoria)

- [ ] **Pitch rate definido**
- [ ] Oxigenación definida (método + timing)
- [ ] Curva térmica definida (setpoints y rampas)
- [ ] Nutrientes definidos (si aplica)
- [ ] **Plan de fermentación completo** (incluye plan de limpieza/diacetilo)

---

## D) Repitch / preservación de cepa (si aplica)

- [ ] ¿Se reutiliza? (sí/no) + generaciones objetivo
- [ ] Plan de cosecha y almacenamiento definido
- [ ] Higiene: limpieza + sanitización entre batches definida

---

## Reglas operativas (B003) — Checklist
<a id="r-0133"></a>

### R-0133 — En cono: cosecha con criterio y descartes para evitar trub/deriva
**Regla:** si cosechás de cono, evita el “fondo sucio” y estandarizá descartes.  
**Evidencia (B003):** “desecha el primer tercio… contiene más trub… células muertas…”.
<a id="r-0134"></a>

### R-0134 — Si hay manejo de cultivo/propagación: la selección limpia es obligatoria
**Regla:** si vas a cultivar/propagar, priorizá evitar contaminantes y mantener consistencia de cepa.  
**Evidencia (B003):** “La selección adecuada de colonias es una parte vital…”.
<a id="r-0135"></a>

### R-0135 — La decisión de cepa debe incluir setpoints coherentes (no sólo “la cepa”)
**Regla:** la salida mínima de “elegí cepa” es: pitch/O2/temperatura/nutrientes coherentes con el objetivo.  
**Evidencia (Parte 5):** “Qué setpoints son coherentes con esa elección (pitch/O2/temperatura/nutrientes)”.
<a id="r-0136"></a>

### R-0136 — La decisión se valida con caso + checklist antes de ejecutar
**Regla:** cerrás la elección cuando pasa checklist y aplica en un caso (Ale Americana / Lager / NEIPA).  
**Evidencia (Parte 5):** “Cómo validar la elección con un caso aplicado y un checklist”.
