# Marco de decisión para selección de cepa

Variables primarias:
- Perfil de azúcares del mosto
- OG / alcohol objetivo
- Temperatura de fermentación disponible
- Perfil sensorial buscado

Variables secundarias:
- Floculación deseada
- Reutilización prevista
- Estabilidad requerida

Regla:
> Elegir cepa es elegir un **comportamiento**, no un nombre.
