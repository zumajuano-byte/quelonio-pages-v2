# Levadura — Parte 5 (Selección de cepas y casos aplicados)

Este bloque integra **mosto + proceso + perfil sensorial** para seleccionar cepas
y diseñar fermentaciones coherentes y reproducibles.
No es un catálogo: es un **marco de decisión**.

---

## Objetivo operativo
Elegir cepa (y diseñar su fermentación) como una decisión sistémica:
**estilo objetivo + receta + proceso + limitaciones reales**.

## Qué vas a poder decidir al final de este bloque
- Qué cepa conviene para un objetivo sensorial (y por qué).
- Qué trade-offs aceptás (limpieza vs carácter, velocidad vs control, floculación vs estabilidad).
- Qué setpoints son coherentes con esa elección (pitch/O2/temperatura/nutrientes).
- Cómo validar la elección con un caso aplicado y un checklist.
