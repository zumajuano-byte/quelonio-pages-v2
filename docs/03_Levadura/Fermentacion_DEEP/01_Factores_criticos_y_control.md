# Fermentación — Factores críticos y control (DEEP)

## 1) Los 3 factores que gobiernan la fermentación
1. **Levadura**: cantidad suficiente + vitalidad (starter o cosecha adecuada).
2. **Mosto**: oxígeno inicial + nutrición (FAN / micronutrientes).
3. **Temperatura**: mantener dentro del rango de cepa y con control estable.

## 2) Oxígeno y nutrición (regla práctica)
- El mosto aporta la mayoría de nutrientes, pero **el oxígeno** es el factor que falta tras hervido.
- La oxigenación es crítica al inicio (fase lag). Evitar oxígeno más tarde salvo casos específicos.

## 3) Temperatura (regla práctica)
- Control de temperatura = palanca #1 de consistencia.
- Desvíos al alza o cambios bruscos incrementan riesgo de precursores de defectos y mala limpieza al final.

## 4) Excepción: cervezas de alta densidad
- En mostos muy altos, puede requerirse oxígeno puro y/o una segunda dosis a las 12–18 horas.
- Ajustar tasa de inoculación y, si aplica, manejo de temperatura para sostener el rendimiento.

## 5) Monitoreo mínimo recomendado (QA/QC)
Orden típico de prioridad:
1) Temperatura
2) Densidad
3) pH
4) Oxígeno disuelto
5) CO₂
