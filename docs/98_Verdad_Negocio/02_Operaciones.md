# Verdad vigente — Operaciones y Procesos

## Propósito
Establecer procesos operativos vigentes (producción, limpieza, empaquetado, frío, despacho) y su estándar mínimo.

## Regla de verdad
- Si un proceso cambia: se actualiza aquí + se registra en `SESSIONS_LOG` (“Actualizaciones de Verdad”).

## Estado
- status: en revisión | vigente
- last_updated: AAAA-MM-DD
- owner: Juan

## Mapa de proceso (alto nivel)
1) Planificación (insumos / agenda / capacidad)
2) Elaboración (molienda → macerado → hervor → whirlpool)
3) Fermentación / maduración
4) Cold crash / clarificación (si aplica)
5) Carbonatación / envasado
6) Empaque / rotulado
7) Almacenamiento / frío
8) Despacho / distribución
