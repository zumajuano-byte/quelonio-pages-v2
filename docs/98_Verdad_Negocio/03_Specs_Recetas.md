# Verdad vigente — Especificaciones de Recetas

## Propósito
Definir qué es “vigente” para cada receta/estilo (parámetros objetivo + tolerancias) para que producción y QA/QC tengan una referencia única.

## Regla de verdad
- Receta “vigente” = la última versión aprobada aquí.
- Cambios experimentales van como “en revisión” y solo pasan a “vigente” si se registra decisión.
