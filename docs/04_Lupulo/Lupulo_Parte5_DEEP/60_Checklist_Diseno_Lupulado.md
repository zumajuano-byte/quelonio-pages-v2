# 60 — Checklist: diseño lupulado

## A) Definición (antes de diseñar)

- [ ] Estilo objetivo y “una frase” sensorial
- [ ] Nivel de amargor objetivo (rango)
- [ ] Nota líder aromática + 2 secundarias

## B) Arquitectura de lupulado (capas)

- [ ] Amargor base: ¿hervor o extracto?
- [ ] Whirlpool/hopstand: temp + tiempo + carga
- [ ] Dry hop: g/L total + etapas + tiempo + temperatura
- [ ] ¿Biotransformación? (sí/no) → timing definido

## C) Riesgos y controles

- [ ] DO/oxígeno: plan de purga y manejo cerrado
- [ ] Hop creep: plan de diacetilo/descanso
- [ ] Vegetal/astringencia: límite de contacto/temperatura y plan de separación
- [ ] Pérdidas: estimación de L perdidos y cómo mitigarlo

## D) Validación

- [ ] Sensorial rápido (nariz/amargor/textura)
- [ ] Estabilidad: ¿se sostiene el aroma?
- [ ] Ajuste para el próximo lote (1 palanca a la vez)
