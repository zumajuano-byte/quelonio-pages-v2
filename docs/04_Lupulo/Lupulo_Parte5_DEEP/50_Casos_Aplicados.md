# 50 — Casos aplicados

## Caso A: West Coast IPA

Objetivo:
- seco, amargor firme, cítrico/resinoso
Decisiones típicas:
- amargor base claro + whirlpool moderado
- dry hop medio con foco en frescura y bajo DO

## Caso B: NEIPA

Objetivo:
- aroma alto tropical, amargor suave, mouthfeel alto
Decisiones típicas:
- whirlpool fuerte + dry hop escalonado
- plan anti hop creep + DO como prioridad

## Caso C: Pils / Lager lupulada (si aplica)

Objetivo:
- aroma delicado, amargor fino
Decisiones típicas:
- minimizar vegetal, maximizar limpieza de proceso y estabilidad
