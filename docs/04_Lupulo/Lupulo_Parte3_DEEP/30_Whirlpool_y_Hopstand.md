# 30 — Whirlpool y hopstand

## 1) Por qué existe esta etapa

Maximiza aroma con menos destrucción que el hervor, pero:
- todavía hay temperatura suficiente para extraer y transformar cosas.

## 2) Variables críticas

- Temperatura de entrada (y su caída real)
- Tiempo de contacto
- Agitación (mecánica)
- Separación (trub + hop matter)

## 3) Regla operable (para no “sobre-extraer”)

Si aparece:
- astringencia / té / verde → bajá temperatura o acortá tiempo
- aroma débil → subí carga o ajustá variedad/timing, pero protegé DO.
