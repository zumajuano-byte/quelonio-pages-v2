# Lúpulo (04)

Este módulo define el lúpulo como **sistema de diseño** (amargor + aroma + estabilidad + costos), no como “ingrediente suelto”.

## Cómo usar este módulo (operativo)
1) Si estás diseñando receta: ir directo a **Parte 5** (marco de diseño).
2) Si estás ajustando amargor/aroma: Parte 2 y Parte 3.
3) Si estás en IPA moderna / NEIPA: Parte 3 y Parte 4 (dry hop, biotransformación, tioles, riesgos).
