# Riesgos y control (IPA moderna)

## Objetivo
Tener un “registro de riesgos” operativo para IPA moderna/NEIPA y controles mínimos reproducibles.

## Riesgos principales (Parte 4)
1) **Hop creep** (re-fermentación / cambios de densidad / potencial diacetilo)
2) **Oxígeno (O₂)** en manipulación post-fermentación (pérdida aromática / oxidación)
3) **Extracción vegetal / astringencia** (polifenoles, contacto excesivo)
4) **Turbidez fuera de objetivo** (estilo vs defecto)
5) **Variabilidad** (timing no definido, sin criterio de corte)

> Nota: el control de O₂ tiene una regla canónica en Parte 3 (Dry hop). Aquí lo tratamos como riesgo transversal del sistema IPA moderna.

<a id="r-0404"></a>
## R-0404 — Hop creep: control por temperatura y tiempo de contacto

**Regla (operativa):**  
Para reducir riesgo de hop creep, definir y controlar explícitamente:
- **Temperatura** de dry hop
- **Tiempo de contacto** (y criterio de corte)

**Evidencia (B006):**  
- FuenteID: B006  
- Ubicación: Cap. 9 (Hop Creep) — *página exacta pendiente*  
- Cita (<=25 palabras): “...se puede controlar ... controlar las temperaturas del dry hop y el tiempo de contacto.”

**Checklist mínimo**
- [ ] Temperatura objetivo definida (rango).
- [ ] Tiempo de contacto objetivo definido (horas/días).
- [ ] Criterio de corte (trasiego/separación).
- [ ] Registro: fecha/hora de adición y retiro; temperatura real.

## Controles mínimos (SOP compacto)
### A) Antes de lupular
- Objetivo sensorial definido + timing definido.
- Plan anti-O₂ (purga / circuito / minimización de aperturas).
- Criterio de corte documentado.

### B) Durante contacto
- Control de temperatura.
- Control de tiempo (no “a ojo”).
- Decisión de separación (sedimentación/trasiego/filtrado si aplica).

### C) Después (verificación)
- Registrar resultados: aroma, astringencia, estabilidad, densidad final si aplica.
- Si hubo cambio inesperado: anotar como “incidente” para mejorar protocolo.
