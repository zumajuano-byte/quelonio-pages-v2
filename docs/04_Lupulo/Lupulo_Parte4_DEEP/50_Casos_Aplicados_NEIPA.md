# Casos aplicados (NEIPA / IPA moderna)

## Objetivo
Bajar a “recetas de decisión” (no recetas numéricas) cómo se usa Parte 4.

## Caso A — NEIPA “tropical” (prioriza intensidad y jugosidad)
**Decisiones**
- Declarar objetivo sensorial (tropical/jugoso).
- Definir si se busca “tioles: sí”.
- Elegir timing con intención (fermentación activa vs post) y justificarlo.
- Diseñar controles (O₂ + hop creep).

**Checklist**
- [ ] Tioles: sí/no declarado.
- [ ] Timing: definido + racional.
- [ ] Plan anti-O₂: escrito y ejecutable.
- [ ] Temperatura + tiempo de contacto definidos (hop creep).

## Caso B — IPA moderna “limpia/definida” (prioriza claridad y estabilidad)
**Decisiones**
- Minimizar extracción vegetal y variabilidad.
- Control fuerte de O₂ post-fermentación.
- Contacto controlado y separación prolija.

**Checklist**
- [ ] Criterio de corte explícito.
- [ ] Separación definida.
- [ ] O₂: controlado de punta a punta.

## Caso C — Ajuste por defecto (cuando aparece un problema)
1) **Astringencia/verde**
   - Revisar tiempo/temperatura de contacto y separación.
2) **Caída rápida de aroma**
   - Auditar O₂ (puntos de entrada en adiciones/trasiegos/envasado).
3) **Cambio de densidad / re-fermentación**
   - Tratar como hop creep: volver a R-0404 y ajustar temperatura/tiempo.
